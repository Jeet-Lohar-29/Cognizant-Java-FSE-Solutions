# Exercise 9 - ES6 Features in React

## Objective

Develop a React application named **cricketapp** to understand and implement various ES6 features used in React.

The application demonstrates:

- map()
- filter()
- Arrow Functions
- Destructuring
- Spread Operator
- Conditional Rendering

## Steps Performed

1. Created the React application `cricketapp`.
2. Created `ListofPlayers.js`.
3. Created `IndianPlayers.js`.
4. Displayed all players using `map()`.
5. Filtered players using `filter()`.
6. Used ES6 arrow functions.
7. Used array destructuring.
8. Merged arrays using the spread operator.
9. Used conditional rendering with a flag variable.
10. Executed the application using `npm start`.

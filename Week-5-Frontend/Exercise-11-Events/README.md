# Exercise 11 - React Event Handling

## Objective

Develop a React application named **eventexamplesapp** to understand React Event Handling.

The application demonstrates:

- Event Handlers
- Multiple Function Calls
- Passing Arguments to Event Handlers
- Synthetic Events
- Currency Conversion using Button Click Events

## Events Implemented

### Increment

- Increases the counter.
- Displays **Hello! Member**.

### Decrement

- Decreases the counter.

### Say Welcome

- Displays **Welcome** using an event handler with an argument.

### Click on Me

- Displays **I was clicked** using a synthetic event.

### Currency Converter

- Converts Indian Rupees into Euro when the Submit button is clicked.

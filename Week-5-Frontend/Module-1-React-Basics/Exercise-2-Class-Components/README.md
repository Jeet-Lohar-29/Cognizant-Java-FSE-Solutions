# Exercise 2 - React Class Components

## Objective

Develop a React application named **StudentApp** using **Class Components**.

Create the following components:

- Home
- About
- Contact

Render all three components in the main App component.

---

- React
- JavaScript (ES6)
- JSX
- Node.js
- npm
- Visual Studio Code

## Steps Performed

1. Created a React application named **StudentApp**.

   - Contact

5. Rendered the components.
6. Executed the application using `npm start`.

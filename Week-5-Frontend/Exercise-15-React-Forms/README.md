# Exercise 15 - React Forms

## Objective

Develop a React application named **ticketraisingapp** to understand the concepts of React Forms and Controlled Components.

The application allows an employee to register a complaint and generates a reference number upon successful submission.

---

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- React Forms
- Controlled Components
- React Hooks (useState)
- Node.js
- npm
- Visual Studio Code

## Features Implemented

- Employee Name Textbox
- Complaint Textarea
- Controlled Components
- Form Handling
- Form Submission
- Random Reference Number Generation
- Alert Confirmation Message

---

## Application Flow

1. Enter Employee Name.
2. Enter Complaint.
3. Click **Submit**.
4. Complaint is registered successfully.
5. A unique Reference Number is generated and displayed in an alert.

## Steps Performed

1. Created the React application `ticketraisingapp`.
2. Created the `ComplaintRegister` component.
3. Implemented controlled input fields.
4. Managed form data using React state.
5. Implemented `handleChange()` for input updates.
6. Implemented `handleSubmit()` for form submission.
7. Generated a random complaint reference number.
8. Displayed confirmation using an alert.
9. Executed the application using `npm start`.

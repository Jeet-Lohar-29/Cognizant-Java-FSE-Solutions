# Week 5 - Frontend (React)

## Cognizant Digital Nurture 4.0 - Java FSE

This repository contains all the hands-on exercises completed during **Week 5** of the Cognizant Digital Nurture 4.0 Java Full Stack Engineer (Java FSE) program.

Week 5 focuses on building modern frontend applications using **React.js**, covering everything from React fundamentals to REST API integration.

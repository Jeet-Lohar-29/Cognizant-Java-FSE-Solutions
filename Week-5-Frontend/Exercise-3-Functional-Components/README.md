# Exercise 3 - React Functional Components

## Objective

Develop a React application named **scorecalculatorapp** using a **Functional Component**.

Create a component named **CalculateScore** that accepts student details through props, calculates the score percentage, and displays the result with CSS styling.

---

- JavaScript (ES6)
- JSX
- CSS
- Node.js
- npm
- Visual Studio Code

1. Created the React application `scorecalculatorapp`.

# Exercise 4 - Functions

## Objective

This exercise demonstrates the creation and use of PL/SQL functions to perform common banking calculations and validations.

---

## Scenarios

### Scenario 1 - Calculate Customer Age

Calculate the age of a customer using the customer's date of birth.

**File**

- `Scenario-1.sql`

---

### Scenario 2 - Calculate Monthly Installment

Calculate the monthly loan installment based on:

- Loan Amount
- Interest Rate
- Loan Duration

**File**

- `Scenario-2.sql`

---

### Scenario 3 - Check Account Balance

Determine whether an account contains sufficient balance before processing a transaction.

**File**

- `Scenario-3.sql`

---

## Concepts Covered

- PL/SQL Functions
- RETURN Statement
- Parameters
- Mathematical Functions
- Conditional Statements
- Exception Handling
- BOOLEAN Return Type

# PL/SQL

## Overview

This folder contains the complete set of **PL/SQL Hands-on Exercises** completed as part of the **Cognizant Digital Nurture 5.0 - Java Full Stack Engineer (Java FSE)** program.

The exercises cover fundamental and advanced PL/SQL concepts, including control structures, exception handling, stored procedures, functions, triggers, cursors, and packages. The solutions are implemented using Oracle Database 21c Express Edition and executed through the Oracle SQL Developer Extension for Visual Studio Code.

## Database Schema

The exercises are based on a banking database consisting of the following tables:

- Customers
- Accounts
- Transactions
- Loans
- Employees

The schema and sample data are created using:

- `CreateTables.sql`
- `InsertSampleData.sql`

These scripts should be executed before running any exercise.

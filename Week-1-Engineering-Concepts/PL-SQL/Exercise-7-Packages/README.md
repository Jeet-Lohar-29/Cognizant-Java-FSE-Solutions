# Exercise 7 - Packages

## Objective

This exercise demonstrates the implementation of PL/SQL packages to group related procedures and functions into reusable modules.

---

## Scenarios

### Scenario 1 - Customer Management Package

Create a package containing:

- Add Customer
- Update Customer
- Get Customer Balance

**File**

- `Scenario-1.sql`

---

### Scenario 2 - Employee Management Package

Create a package containing:

- Hire Employee
- Update Employee Details
- Calculate Annual Salary

**File**

- `Scenario-2.sql`

---

### Scenario 3 - Account Operations Package

Create a package containing:

- Open Account
- Close Account
- Get Total Customer Balance

**File**

- `Scenario-3.sql`

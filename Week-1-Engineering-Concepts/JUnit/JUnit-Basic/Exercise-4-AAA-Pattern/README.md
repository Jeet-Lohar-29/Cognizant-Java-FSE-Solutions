# Exercise 4 - Arrange-Act-Assert (AAA) Pattern, Test Fixtures, Setup and Teardown

## Objective

Learn to organize JUnit test cases using the Arrange-Act-Assert (AAA) pattern and use setup and teardown methods.

---

## Scenario

Create a Calculator class and write unit tests following the AAA pattern. Use `@Before` to initialize the test fixture and `@After` to clean up after each test.

---

- pom.xml
- Calculator.java
- CalculatorTest.java

# JUnit Basic

## Overview

This folder contains the **JUnit Basic** exercises completed as part of the **Cognizant Digital Nurture 5.0 - Java Full Stack Engineer (Java FSE)** program.

The exercises introduce the fundamentals of unit testing using **JUnit 4**, including project setup with Maven, writing basic test cases, using assertions, and organizing tests with the Arrange-Act-Assert (AAA) pattern along with setup and teardown methods.

## Exercises

### Exercise 1 – Setting Up JUnit

**Objective**

Configure a Maven-based Java project with JUnit 4.13.2 and create the first unit test.

**Concepts Covered**

* Maven Project Setup
* JUnit Dependency
* pom.xml Configuration
* First Test Case

---

### Exercise 2 – Writing Basic JUnit Tests

**Objective**

Create a simple Java class and write unit tests for its methods.

**Concepts Covered**

* @Test Annotation
* assertEquals()
* Unit Testing
* Multiple Test Methods

---

### Exercise 3 – Assertions in JUnit

**Objective**

Learn how to validate expected results using JUnit assertion methods.

**Assertions Used**

* assertEquals()
* assertTrue()
* assertFalse()
* assertNull()
* assertNotNull()

---

### Exercise 4 – Arrange-Act-Assert (AAA) Pattern

**Objective**

Organize test cases using the AAA pattern and implement setup and teardown methods.

**Concepts Covered**

* Arrange-Act-Assert (AAA)
* Test Fixtures
* @Before
* @After

---

## Build and Run

Open the terminal inside any exercise folder and execute:

```bash
mvn clean test
```

A successful execution should display:

```text
BUILD SUCCESS
```

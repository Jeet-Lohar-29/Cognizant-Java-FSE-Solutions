# JUnit

## Overview

This folder contains all **JUnit** and **Spring Testing** exercises completed as part of the **Cognizant Digital Nurture 5.0 – Java FSE** program.

The exercises cover the fundamentals and advanced concepts of unit testing using **JUnit 5**, along with testing Spring Boot applications using **Mockito**, **MockMvc**, **Spring Boot Test**, and **Spring Data JPA**.

JUnit 5 consists of three major components: **JUnit Platform**, **JUnit Jupiter**, and **JUnit Vintage**, providing a modern framework for writing and executing automated tests. :contentReference[oaicite:0]{index=0}

# Exercise 9 - Parameterized Test with JUnit

## Objective

Use JUnit 5 parameterized tests to validate multiple inputs using a single test method.

---

## Scenario

Create a simple `EvenChecker` class and test multiple even numbers using `@ParameterizedTest` and `@ValueSource`.

---

* SpringTestApplication.java
* EvenChecker.java
* EvenCheckerTest.java

# Exercise 4 - Integration Test with Spring Boot

## Objective

Write an integration test that verifies the complete flow from the REST controller to the database.

---

## Scenario

Test a REST endpoint that retrieves a user by ID using the complete Spring Boot application context.

---

* UserIntegrationTest.java

* REST API Testing

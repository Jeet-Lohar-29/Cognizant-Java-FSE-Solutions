# Spring Test

## Overview

This folder contains the Spring Testing exercises completed as part of the **Cognizant Digital Nurture 5.0 – Java FSE** program.

The exercises demonstrate how to write and execute unit tests, integration tests, controller tests, repository tests, and parameterized tests using the Spring Boot testing framework.

---

## Exercises

### Exercise 1 – Service Test
- Unit testing a Spring service
- `@SpringBootTest`
- JUnit 5 Assertions

### Exercise 2 – Repository Mocking
- Mocking Spring Data repositories
- Mockito
- `@MockBean`

### Exercise 3 – MockMvc
- Testing REST controllers
- MockMvc
- `@WebMvcTest`

### Exercise 4 – Integration Test
- End-to-end Spring Boot testing
- H2 In-Memory Database
- Spring Data JPA
- `@SpringBootTest`

### Exercise 5 – POST Endpoint Test
- Testing REST POST APIs
- JSON Request Body
- ObjectMapper
- MockMvc

### Exercise 6 – Service Exception Test
- Exception handling in services
- Mockito
- `assertThrows()`

### Exercise 7 – Custom Repository Query
- Spring Data JPA
- Derived Query Methods
- Repository Testing
- `@DataJpaTest`

### Exercise 8 – Controller Advice
- Global Exception Handling
- `@ControllerAdvice`
- `@ExceptionHandler`
- MockMvc Testing

### Exercise 9 – Parameterized Test
- JUnit 5 Parameterized Tests
- `@ParameterizedTest`
- `@ValueSource`

After completing these exercises, you will understand how to:

- Write unit tests for Spring services
- Mock dependencies using Mockito

- Test Spring Data JPA repositories

- Write reusable parameterized JUnit tests
- Build reliable and maintainable test suites

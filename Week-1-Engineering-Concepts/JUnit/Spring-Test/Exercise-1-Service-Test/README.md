# Exercise 1 - Basic Unit Test for a Service Method

## Objective

Write a basic Spring Boot unit test for a service method using JUnit 5.

---

## Scenario

Create a `CalculatorService` with an `add()` method and verify its output using a Spring Boot test.

---

* SpringTestApplication.java
* CalculatorService.java

* Assertions

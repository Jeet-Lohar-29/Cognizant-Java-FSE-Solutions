# Exercise 2 - Mocking a Repository in a Service Test

## Objective

Test a Spring service that depends on a repository by mocking the repository using Mockito.

---

## Scenario

Create a `UserService` that fetches a user from a `UserRepository`. Write a unit test that mocks the repository and verifies the service logic.

---

* pom.xml
* SpringTestApplication.java

* UserRepository.java
* UserService.java
* UserServiceTest.java

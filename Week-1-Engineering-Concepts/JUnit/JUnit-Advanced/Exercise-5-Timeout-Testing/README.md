# Exercise 5 - Timeout and Performance Testing

## Objective

Learn how to verify that a method completes execution within a specified time limit using JUnit 5.

---

## Scenario

Create a `PerformanceTester` class and write a unit test to ensure its `performTask()` method completes within the expected timeout.

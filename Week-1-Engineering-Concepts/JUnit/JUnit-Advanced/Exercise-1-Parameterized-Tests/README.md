# Exercise 1 - Parameterized Tests

## Objective

Learn how to write parameterized tests using JUnit 5 to execute the same test with multiple input values.

---

## Scenario

Create an `EvenChecker` class and verify its `isEven()` method using parameterized tests.

---

## Files

* pom.xml
* EvenChecker.java
* EvenCheckerTest.java

---

## Concepts Covered

* JUnit 5
* Parameterized Tests
* @ParameterizedTest
* @ValueSource
* Assertions
* Maven

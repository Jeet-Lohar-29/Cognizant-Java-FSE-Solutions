# Exercise 3 - Test Execution Order

## Objective

Learn how to control the execution order of JUnit 5 test methods using the `@TestMethodOrder` and `@Order` annotations.

---

## Scenario

Create a test class containing multiple test methods and execute them in a predefined order.

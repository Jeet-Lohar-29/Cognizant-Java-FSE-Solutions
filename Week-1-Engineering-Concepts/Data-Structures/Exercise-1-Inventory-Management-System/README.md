# Exercise 1 - Inventory Management System

## Objective

Develop an Inventory Management System using an efficient data structure.

## Scenario

A warehouse inventory requires efficient storage, retrieval, updating, and deletion of product information.

## Data Structure Used

- HashMap<Integer, Product>

## Files

- Product.java
- InventoryManager.java
- InventoryManagementDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java InventoryManagementDemo
```

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) Average |
| Update | O(1) Average |
| Delete | O(1) Average |
| Search | O(1) Average |

## Learning Outcome

- HashMap
- CRUD Operations
- Time Complexity Analysis
- Inventory Management

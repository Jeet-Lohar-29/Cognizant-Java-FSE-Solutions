# Exercise 6 - Library Management System

## Objective

Implement Linear Search and Binary Search to search books by title.

## Scenario

A library system should allow users to search books efficiently using different searching algorithms.

## Data Structure Used

- Array of Book Objects

## Files

- Book.java
- LibrarySearch.java
- LibraryManagementDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java LibraryManagementDemo
```

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|---------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Learning Outcome

- Linear Search
- Binary Search
- Arrays
- Searching Algorithms
- Time Complexity Analysis

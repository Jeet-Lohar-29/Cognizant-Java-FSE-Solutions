# Exercise 5 - Task Management System

## Objective

Implement a Task Management System using a Singly Linked List.

## Scenario

Manage tasks efficiently by supporting add, search, traverse, and delete operations.

## Data Structure Used

- Singly Linked List

## Files

- Task.java
- TaskManagementSystem.java
- TaskManagementDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java TaskManagementDemo
```

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(n) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Learning Outcome

- Singly Linked List
- Dynamic Data Structure
- CRUD Operations
- Time Complexity Analysis

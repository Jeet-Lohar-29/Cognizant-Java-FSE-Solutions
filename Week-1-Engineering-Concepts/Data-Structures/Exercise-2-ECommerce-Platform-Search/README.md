# Exercise 2 - E-Commerce Platform Search

## Objective

Implement and compare Linear Search and Binary Search algorithms for product searching.

## Scenario

An e-commerce platform needs an efficient search feature for locating products.

## Data Structure Used

- Array of Product Objects

## Files

- Product.java
- SearchAlgorithms.java
- SearchDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java SearchDemo
```

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|---------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Learning Outcome

- Big O Notation
- Linear Search
- Binary Search
- Arrays
- Time Complexity Analysis

# Exercise 4 - Employee Management System

## Objective

Implement an Employee Management System using arrays.

## Scenario

Manage employee records efficiently by supporting add, search, traverse, and delete operations.

## Data Structure Used

- Array of Employee Objects

## Files

- Employee.java
- EmployeeManagementSystem.java
- EmployeeManagementDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java EmployeeManagementDemo
```

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Learning Outcome

- Arrays
- CRUD Operations
- Time Complexity Analysis
- Employee Record Management

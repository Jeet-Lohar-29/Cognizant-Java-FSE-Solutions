# Exercise 3 - Sorting Customer Orders

## Objective

Implement Bubble Sort and Quick Sort to sort customer orders based on total price.

## Scenario

An e-commerce platform needs to prioritize high-value customer orders by sorting them according to their total purchase amount.

## Data Structure Used

- Array of Order Objects

## Files

- Order.java
- SortingAlgorithms.java
- SortingDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java SortingDemo
```

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|---------|-------|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |

## Learning Outcome

- Bubble Sort
- Quick Sort

- Sorting Algorithms

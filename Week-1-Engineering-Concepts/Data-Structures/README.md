# Data Structures & Algorithms

This folder contains the implementation of the **Data Structures & Algorithms (DSA)** exercises provided as part of the **Cognizant Digital Nurture 5.0 - Java Full Stack Engineer (Java FSE)** hands-on program.

The exercises are implemented using **Core Java** with simple, clean, and easy-to-understand code. Each exercise demonstrates the practical application of fundamental data structures, searching algorithms, sorting algorithms, recursion, and complexity analysis.

---

## Objectives

- Understand fundamental Data Structures and Algorithms.
- Analyze the efficiency of algorithms using Big O Notation.
- Implement searching, sorting, recursion, and linked list operations.
- Compare different algorithms based on their performance.
- Develop practical problem-solving skills using Java.

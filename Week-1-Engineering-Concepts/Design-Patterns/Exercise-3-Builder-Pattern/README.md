# Exercise 3 - Builder Pattern

## Objective

Implement the Builder Design Pattern to create complex Computer objects with optional attributes.

## Scenario

A computer can have different configurations such as CPU, RAM, Storage, and Graphics Card. The Builder Pattern allows creating different configurations in a readable and flexible way.

## Files

- Computer.java
- BuilderPatternDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java BuilderPatternDemo
```

## Design Pattern Used

Builder Pattern

## Learning Outcome

- Static Nested Class
- Method Chaining
- Private Constructor
- Builder Design Pattern

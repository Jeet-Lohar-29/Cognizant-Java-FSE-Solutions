# Exercise 4 - Adapter Pattern

## Objective

Implement the Adapter Design Pattern to integrate different third-party payment gateways with a common interface.

## Scenario

A payment processing system needs to work with multiple payment gateways, each having a different interface.

## Files

- PaymentProcessor.java
- PayPalGateway.java
- StripeGateway.java
- PayPalAdapter.java
- StripeAdapter.java
- AdapterPatternDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java AdapterPatternDemo
```

## Design Pattern Used

Adapter Pattern

## Learning Outcome

- Interfaces
- Composition
- Third-party API Integration
- Adapter Design Pattern

# Exercise 2 - Factory Method Pattern

## Objective

Implement the Factory Method Design Pattern to create different types of documents without directly instantiating their classes.

## Scenario

A document management system needs to create different document types such as Word, PDF, and Excel.

## Files

- Document.java
- WordDocument.java
- PdfDocument.java
- ExcelDocument.java
- DocumentFactory.java
- WordDocumentFactory.java
- PdfDocumentFactory.java
- ExcelDocumentFactory.java
- FactoryMethodDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run
```bash
java FactoryMethodDemo
```

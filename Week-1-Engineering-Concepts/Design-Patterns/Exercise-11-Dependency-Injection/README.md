# Exercise 11 - Dependency Injection

## Objective

Implement Dependency Injection using Constructor Injection.

## Scenario

A customer management application should inject a repository into a service class instead of creating it directly.

## Files

- CustomerRepository.java
- CustomerRepositoryImpl.java
- CustomerService.java
- DependencyInjectionDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java DependencyInjectionDemo
```

## Design Pattern Used

Dependency Injection (Constructor Injection)

## Learning Outcome

- Dependency Injection
- Constructor Injection
- Interfaces
- Loose Coupling
- Better Testability

# Week 1 - Engineering Concepts

# Design Patterns

This folder contains the implementation of the **Design Pattern** exercises provided as part of the **Cognizant Digital Nurture 5.0 Java Full Stack Engineer (Java FSE)** hands-on program.

The exercises are implemented using **Core Java** with simple, clean, and easy-to-understand code. Each exercise demonstrates a specific design pattern along with a small real-world scenario.

---

## Objectives

- Understand the purpose of commonly used Design Patterns.
- Learn how Design Patterns improve software design.
- Implement each pattern using Java.
- Demonstrate real-world use cases of Object-Oriented Design Principles.

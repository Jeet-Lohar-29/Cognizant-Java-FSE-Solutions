# Exercise 8 - Strategy Pattern

## Objective

Implement the Strategy Design Pattern to support multiple payment methods that can be selected at runtime.

## Scenario

A payment system allows users to choose different payment methods such as Credit Card and PayPal.

## Files

- PaymentStrategy.java
- CreditCardPayment.java
- PayPalPayment.java
- PaymentContext.java
- StrategyPatternDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java StrategyPatternDemo
```

## Design Pattern Used

Strategy Pattern

## Learning Outcome

- Interfaces
- Runtime Polymorphism
- Composition
- Behavioral Design Pattern

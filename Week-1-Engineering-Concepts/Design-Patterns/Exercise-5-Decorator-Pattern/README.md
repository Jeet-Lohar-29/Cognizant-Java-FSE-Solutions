# Exercise 5 - Decorator Pattern

## Objective

Implement the Decorator Design Pattern to dynamically add notification channels.

## Scenario

A notification system should be able to send notifications through Email, SMS, and Slack without modifying the original EmailNotifier class.

## Files

- Notifier.java
- EmailNotifier.java
- NotifierDecorator.java
- SMSNotifierDecorator.java
- SlackNotifierDecorator.java
- DecoratorPatternDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java DecoratorPatternDemo
```

## Design Pattern Used

Decorator Pattern

## Learning Outcome

- Interfaces
- Abstract Classes
- Composition
- Dynamic Behavior Extension
- Decorator Design Pattern

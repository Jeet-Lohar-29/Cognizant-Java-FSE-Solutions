# Exercise 6 - Proxy Pattern

## Objective

Implement the Proxy Design Pattern to add lazy initialization and caching for image loading.

## Scenario

An image viewer application loads images from a remote server. Using a proxy ensures that images are loaded only when required and reused for subsequent requests.

## Files

- Image.java
- RealImage.java
- ProxyImage.java
- ProxyPatternDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java ProxyPatternDemo
```

## Design Pattern Used

Proxy Pattern (Virtual Proxy)

- Interfaces
- Lazy Initialization

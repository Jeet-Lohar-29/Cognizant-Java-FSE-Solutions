# Exercise 9 - Command Pattern

## Objective

Implement the Command Design Pattern to control a home automation system.

## Scenario

A remote control should issue commands to turn a light ON or OFF without directly interacting with the Light class.

## Files

- Command.java
- Light.java
- LightOnCommand.java
- LightOffCommand.java
- RemoteControl.java
- CommandPatternDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java CommandPatternDemo
```

## Design Pattern Used

Command Pattern

## Learning Outcome

- Command Interface
- Receiver
- Invoker
- Loose Coupling
- Behavioral Design Pattern

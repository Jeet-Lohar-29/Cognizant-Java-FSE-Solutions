# Exercise 7 - Observer Pattern

## Objective

Implement the Observer Design Pattern to notify multiple clients whenever stock prices change.

## Scenario

A stock market monitoring system should notify registered clients (Mobile App and Web App) whenever a stock price is updated.

## Files

- Observer.java
- Stock.java
- StockMarket.java
- MobileApp.java
- WebApp.java
- ObserverPatternDemo.java

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java ObserverPatternDemo
```

## Design Pattern Used

Observer Pattern

## Learning Outcome

- One-to-Many Relationship
- Loose Coupling
- Event Notification
- Interfaces
- Observer Design Pattern

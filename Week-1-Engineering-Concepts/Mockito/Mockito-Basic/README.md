# Mockito-Basic

## Overview

This folder contains the **Mockito Basic** exercises completed as part of the **Cognizant Digital Nurture 5.0 – Java FSE** program.

These exercises introduce the core concepts of Mockito, including creating mock objects, stubbing methods, verifying interactions, argument matching, handling void methods, verifying invocation order, and testing exception scenarios. Mockito enables isolated unit testing by replacing real dependencies with configurable mock objects.

---

## Exercises

### Exercise 1 – Mocking and Stubbing

* Create mock objects
* Stub methods using `when()` and `thenReturn()`

### Exercise 2 – Verifying Interactions

* Verify method invocations
* `verify()`

### Exercise 3 – Argument Matching

* Match arguments using Mockito matchers
* `anyString()`

### Exercise 4 – Handling Void Methods

* Mock void methods
* `doNothing()`
* Verify interactions

### Exercise 5 – Multiple Return Values

* Consecutive stubbing
* Multiple `thenReturn()` values

### Exercise 6 – Verifying Interaction Order

* Verify method invocation sequence
* `InOrder`

### Exercise 7 – Void Methods with Exceptions

* Mock exceptions for void methods
* `doThrow()`
* `assertThrows()`

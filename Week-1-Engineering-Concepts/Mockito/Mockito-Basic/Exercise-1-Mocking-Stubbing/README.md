# Exercise 1 - Mocking and Stubbing

## Objective

Learn the basics of Mockito by creating mock objects and stubbing their behavior.

---

## Scenario

A service depends on an external API. Instead of calling the real API, Mockito is used to create a mock object and define its return value.

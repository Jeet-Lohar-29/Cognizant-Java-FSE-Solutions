# Week 4 - Sample Microservices Exercises

This module covers the fundamentals of **Centralized Authentication and Single Sign-On (SSO)** using **Spring Boot 3**, **Spring Security**, **OAuth2**, **OpenID Connect (OIDC)** and **JSON Web Tokens (JWT)**.

The exercises introduce modern authentication mechanisms that are widely used in Microservices Architecture to secure REST APIs and provide centralized identity management.

---

# 📖 Module Overview

This module consists of three hands-on exercises:

### ✅ Exercise 1 - OAuth2 / OpenID Connect (OIDC)

Implemented centralized authentication using Google OAuth2.

**Topics Covered**

- Spring Security
- OAuth2 Client
- Google Authentication
- OpenID Connect (OIDC)
- SecurityFilterChain

---

### ✅ Exercise 2 - Authorization Server & Resource Server

Configured a Resource Server to protect REST APIs using Spring Security.

**Topics Covered**

- OAuth2 Resource Server
- JWT Validation
- Protected REST APIs
- HTTP 401 Unauthorized
- Security Configuration

---

### ✅ Exercise 3 - JWT Authentication

Implemented JWT-based authentication and authorization for REST APIs.

**Topics Covered**

- JWT Generation
- JWT Validation
- JWT Filter
- Stateless Authentication
- Protected Endpoints

---

# 🎯 Objectives

- Understand centralized authentication.
- Configure OAuth2 Login.

# Spring Core & Maven

This folder contains all hands-on exercises completed as part of the **Cognizant DN 5 Java FSE Deep Skilling Program** under the **Spring Core & Maven** module.

The exercises cover the fundamentals of the Spring Framework, including Inversion of Control (IoC), Dependency Injection (DI), Aspect-Oriented Programming (AOP), Maven project configuration, annotation-based bean management, and the development of a Spring Boot REST application using Spring Data JPA and an H2 in-memory database.

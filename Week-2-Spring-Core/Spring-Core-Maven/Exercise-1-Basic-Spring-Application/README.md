# Exercise 1 - Configuring a Basic Spring Application

## Objective

Learn how to create a basic Spring application using XML configuration and the Spring IoC Container.

---

## Problem Statement

Develop a simple Library Management application.

The application demonstrates:

- Spring XML Configuration
- Bean Definition
- Bean Creation
- Loading the Spring Context

---

## Technologies Used

- Java
- Spring Core
- Maven

---

## Project Structure

```
Exercise-1-Basic-Spring-Application
│
├── src
│
├── pom.xml
│
└── README.md
```

---

## Files Description

### BookRepository.java

Repository Bean.

### BookService.java

Service Bean.

### applicationContext.xml

Spring Bean Configuration.

### LibraryManagementApplication.java

Loads the Spring Container and retrieves beans.

---

## Expected Output

```
BookRepository Bean Created
BookService Bean Created

Spring Container Started Successfully
```

---

## Concepts Used

- Spring Core
- Spring IoC
- Bean Configuration
- XML Configuration
- ApplicationContext
- ClassPathXmlApplicationContext

---

## Learning Outcomes

After completing this exercise, you will understand:

- What Spring IoC is.
- How Spring creates beans.
- XML-based configuration.
- Loading the Spring container.
- Retrieving beans from the container.

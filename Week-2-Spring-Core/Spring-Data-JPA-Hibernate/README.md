# Spring Data JPA & Hibernate

## Cognizant Digital Nurture 4.0 - Java Full Stack Engineer

## Overview

This repository contains all the hands-on exercises completed for the **Spring Data JPA & Hibernate** module of the Cognizant Digital Nurture 4.0 program.

The module covers Hibernate fundamentals, Spring Data JPA, ORM mapping, query methods, HQL, Criteria API, repository pattern, auditing, projections, pagination, datasource configuration, and Hibernate-specific features.

Spring Data JPA simplifies data access by reducing boilerplate code while Hibernate provides the JPA implementation and ORM capabilities for Java applications. :contentReference[oaicite:0]{index=0}

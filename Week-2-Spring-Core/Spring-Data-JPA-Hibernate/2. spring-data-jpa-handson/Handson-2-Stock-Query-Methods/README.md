# Handson-2: Stock Query Methods

## Objective

Implement and demonstrate **Spring Data JPA Derived Query Methods** using a Stock entity.

This hands-on focuses on creating repository methods whose SQL queries are automatically generated by Spring Data JPA based on method names.

## Repository Query Methods

Implemented derived query methods:

```java
findByCodeStartingWith(String prefix)

findByNameContaining(String keyword)

findByNameContainingOrderByNameAsc(String keyword)
```

Spring Data JPA automatically derives SQL queries from these method names without requiring explicit SQL or JPQL. :contentReference[oaicite:0]{index=0}

---

## Service Methods

- Search stocks by code prefix
- Search stocks containing a keyword
- Search stocks containing a keyword sorted alphabetically

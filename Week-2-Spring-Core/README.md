# Week-2: Spring Core

## Cognizant Digital Nurture 4.0 - Java Full Stack Engineer

## Overview

This repository contains all the hands-on exercises completed during **Week-2: Spring Core** of the Cognizant Digital Nurture 4.0 Java Full Stack Engineer learning path.

The week focuses on **Spring Framework**, **Spring Boot**, **Spring Data JPA**, and **Hibernate ORM**, covering dependency injection, aspect-oriented programming, repository development, ORM mapping, query methods, CRUD operations, auditing, projections, pagination, and Hibernate features.

Spring Boot simplifies enterprise application development through auto-configuration, while Spring Data JPA significantly reduces boilerplate code for data access using repositories. :contentReference[oaicite:0]{index=0}
